# Placement Prediction using Machine Learning

## Project Overview

This project predicts whether a student will get placed or not based on academic performance, communication skills, internships, projects, aptitude score, technical skills, and IQ using Machine Learning classification algorithms.

This is a strong college-level ML project that is highly useful for:
- Final Year Project
- Minor Project
- Resume Building
- GitHub Portfolio
- Placement Interviews

---

## Features

- Data Preprocessing
- Feature Selection
- Feature Scaling
- Multiple Classification Algorithms
- Accuracy Comparison
- Precision, Recall, F1 Score
- Confusion Matrix
- Classification Report
- Best Model Selection
- Data Visualization using Graphs

---

## Dataset Features

The dataset contains the following columns:

| Feature | Description |
|---|---|
| CGPA | Student academic performance |
| IQ | Intelligence Quotient |
| Communication | Communication skills score |
| Internships | Number of internships completed |
| Projects | Number of projects completed |
| Aptitude | Aptitude test score |
| TechnicalSkills | Technical skills rating |
| Placed | Target variable (0 = No, 1 = Yes) |
